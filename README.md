## Төслийн зорилго

Энэхүү төсөл нь Объект Хандлагат Программчлалын (OOP) үндсэн зарчмууд болон бодит цагийн тоглоомын гогцоо (Game Loop)-ийн логикийг практик дээр хэрэгжүүлэх зорилготой. Систем нь хэрэглэгчийн гарнаас шивэх хурд (WPM) болон нарийвчлалыг шалгах бөгөөд санах ойн ашиглалтыг оновчтой болгох үүднээс хэш кэшлэх (Memoization) болон шугаман хайлтын алгоритмуудыг ашигласан.

## Гол боломжууд

* **Хэлний сонголт:** Үндсэн цэс (Main Menu) дээр Англи болон Монгол хэлийг сонгох боломжтой.
* **Тоглоомын механик:** Дэлгэцийн дээрээс унах үгсийг алдаагүй шивж лазераар устгах үндсэн логик.
* **Комбо систем:** Алдаагүй дараалж шивэх тусам Combo нэмэгдэж, оноо үржигдэнэ.
* **Хүндрэлийн үе шат:** 4-р түвшин буюу Level 4-өөс эхлэн дайсны үгнүүд хөлөг рүү сум (Projectile) буудах бөгөөд тоглогч сумыг түрүүлж шивж устгах шаардлагатай.
* **График эффектүүд:** CRT дэлгэцийн эффект, неон гэрэлтэлт (Neon Glow), дэлгэц чичрэх (Screen Shake), бөөмийн систем (Particle System).
* **Алдааны сануулга:** Буруу үсэг дарахад дэлгэцийн ирмэг улаанаар анивчих (Wrong Flash) эффект.
* **Онооны бүртгэл:** Хамгийн өндөр оноог `highscore.json` файлд автоматаар хадгалж, дараагийн тоглолтод уншина.

## Ашигласан OOP болон Программчлалын ойлголтууд

* **Класс болон Объектууд:** Тоглоомын бүх элементийг бие даасан класс болгон зохион байгуулсан (`Game`, `MainMenu`, `Word`, `Spaceship`, `Projectile`, `UI`, `ParticleSystem`).
* **Ассоциаци ба Комппозиц холбоо:** `Game` класс нь унаж буй үгс (`Word`) болон сумнуудыг (`Projectile`) динамик жагсаалт (List) хэлбэрээр дотроо агуулж, удирдах холбоо.
* **Хэш хүснэгт (Dictionary):** Неон гэрэлтэлттэй текстийг секунд бүр дахин зурж FPS унагаахаас сэргийлж `_neon_cache` толь бичигт урьдчилан зурагдсан гадаргууг кэшлэх (Memoization) арга.
* **Шугаман хайлт (Linear Search):** Тоглогч үсэг дарах үед дэлгэц дээрх идэвхтэй үгсээс тохирох үгийг хайж олох логик.
* **Файл I/O болон JSON:** Тоглоомын дээд оноог файлаас унших болон файл руу бичих үйлдэл.
